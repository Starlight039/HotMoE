# Hybrid Routing for a Mixture of LoRA Experts

## Overview
This repository contains the implementation of "Hybrid Routing for a Mixture of LoRA Experts", a novel multi-task in-
struction tuning framework that adapts hierarchical routing to
the distinct characteristics of different LLM layers.
